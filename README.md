# blender-character-animation-lab
Description

A Blender-based 3D character animation project focused on expressive motion, storytelling, and reusable animation workflows.

This repository contains animated character scenes, rigging experiments, motion tests, and cinematic animation setups used across creative projects.

Built for collaboration, iteration, and real production use.
